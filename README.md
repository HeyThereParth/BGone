# BGone
This is a simple web-based tool that removes the background of an image and converts it to a PNG format with transparency. It is built using HTML, CSS, and JavaScript, providing a user-friendly interface for seamless interaction.

## Features  
- Upload any image file.  
- Automatically removes the background from the image.  
- Converts the image to PNG format with a transparent background.  
- Fast and efficient processing.  

## Technologies Used  
- **HTML**: For structuring the web page.  
- **CSS**: For styling and responsive design.  
- **JavaScript**: For handling image processing and functionality.  

## How to Use  
1. Clone or download this repository.  
2. Open `index.html` in your web browser.  
3. Upload your image using the provided file input.  
4. Download the processed image as a PNG with the background removed.
